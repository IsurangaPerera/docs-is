## Prerequisite

To run the project on local, it requires [python](https://www.python.org/downloads/) & [mkdocs](https://www.mkdocs.org/)

## Run project locally (Dev Mode)

1. Clone repo
2. Run `mkdocs serve` command in `<Lang folder>`. E.g. `cd en && mkdocs serve`
